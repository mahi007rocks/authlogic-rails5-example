# README

[authlogic](https://github.com/binarylogic/authlogic) minimal example via rails 5.

# Features

* Login
   * `/login`
* Logout
   * `/logout`
* Signup
   * `/signup`
* User model CRUD
   * The operation can be controled after login.

# Getting started

```
bin/setup
bin/rails s
```

# Test

```
bin/rake
```
